# Time-Table-Manager

# Technology-Used

- MongoDB
- Express.js
- React.js
- Node.js


# How_To_Run

- ## How To run backend 

```
$ git clone https://github.com/Saurabhbiswas16/Time-Table-Manager
$ cd Time-Table-Manager

$ cd calendar-todo-backend
$ npm install

$ node server.js
```

- ## How To run Front-end(React)

```
$ git clone https://github.com/Saurabhbiswas16/Time-Table-Manager
$ cd Time-Table-Manager

$ cd calendar-todo-frontend
$ npm install

$ npm start 
```
